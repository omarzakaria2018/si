الترقيم الذكي

  • الهدف: ترقيم الأزرار الظاهرة فقط بدءاً من رقم 1
  • الذكاء: تجاهل الأزرار المخفية تماماً
  • الديناميكية: إعادة حساب الأرقام عند كل فتح للقائمة
  • التقنية: JavaScript بدلاً من CSS Counter

المشكلة والحل

❌ المشكلة السابقة

  • CSS Counter يعد جميع العناصر
  • الأزرار المخفية تأخذ أرقام
  • ترقيم غير متسلسل للظاهرة
  • مثال: 1، 3، 5، 8، 10

✅ الحل الذكي

  • JavaScript يفحص العناصر الظاهرة
  • الأزرار المخفية لا تأخذ أرقام
  • ترقيم متسلسل للظاهرة فقط
  • مثال: 1، 2، 3، 4، 5

الكود الذكي

🧠 دالة الترقيم الذكي:

function addNumbersToVisibleMenuItems() { const menuList = document.querySelector('.mobile-dropdown-list'); // إزالة الأرقام السابقة const allButtons = menuList.querySelectorAll('li > button'); allButtons.forEach(button => { const existingNumber = button.querySelector('.menu-number'); if (existingNumber) existingNumber.remove(); }); // الحصول على العناصر الظاهرة فقط const visibleItems = []; const allItems = menuList.querySelectorAll('li'); allItems.forEach(item => { const button = item.querySelector('button'); if (!button) return; // التحقق من أن العنصر ظاهر const isHidden = item.style.display === 'none' || getComputedStyle(item).display === 'none' || button.style.display === 'none' || getComputedStyle(button).display === 'none'; if (!isHidden) { visibleItems.push({ item, button }); } }); // إضافة الأرقام للعناصر الظاهرة visibleItems.forEach((itemData, index) => { const { button } = itemData; const number = index + 1; // بدءاً من 1 // إنشاء عنصر الرقم const numberElement = document.createElement('span'); numberElement.className = 'menu-number'; numberElement.textContent = number + '. '; // تنسيق جميل للرقم... button.insertBefore(numberElement, button.firstChild); }); }

الميزات الذكية

الذكاء المطبق:

  • فحص الرؤية: يتحقق من display: none و getComputedStyle
  • إزالة السابق: ينظف الأرقام السابقة قبل إضافة جديدة
  • ترقيم متسلسل: يبدأ من 1 للعناصر الظاهرة فقط
  • تحديث ديناميكي: يعيد الحساب عند كل فتح للقائمة
  • تصميم جميل: أرقام ملونة مع خلفية دائرية

اختبار الترقيم الذكي

خطوات الاختبار

📋 للاختبار الفعلي:

  1. افتح الصفحة الرئيسية (index.html)
  2. افتح Console (F12) لمراقبة الرسائل
  3. قلل حجم الشاشة لأقل من 900px
  4. انقر على زر القائمة (☰) في الهيدر
  5. تحقق من الترقيم:
    • ✅ الأزرار الظاهرة مرقمة: 1، 2، 3، 4، 5...
    • ❌ الأزرار المخفية لا تأخذ أرقام
    • ✅ ترقيم متسلسل بدون فجوات
    • ✅ رسالة في Console تؤكد عدد الأزرار المرقمة
  6. اختبر مع مستخدمين مختلفين:
    • بعض الأزرار مخفية حسب صلاحيات المستخدم
    • الترقيم يتكيف مع الأزرار الظاهرة فقط

🔍 رسائل Console المتوقعة:

🔢 بدء ترقيم الأزرار الظاهرة في القائمة المحمولة... ✅ تم ترقيم 8 زر ظاهر في القائمة المحمولة

النتائج المتوقعة

🎯 ما ستراه:

  • ترقيم متسلسل: 1، 2، 3، 4، 5، 6، 7، 8...
  • بدون فجوات: لا توجد أرقام مفقودة
  • أزرار ظاهرة فقط: المخفية لا تؤثر على الترقيم
  • تحديث ديناميكي: إعادة حساب عند كل فتح
  • تصميم جميل: أرقام ملونة مع خلفية
