كود حذف الأزرار الفارغة

🗑️ دالة حذف الأزرار الفارغة:

function removeEmptyMenuButtons() { console.log('🗑️ حذف الأزرار الفارغة من القائمة المحمولة...'); const menuList = document.querySelector('.mobile-dropdown-list'); const allItems = menuList.querySelectorAll('li'); let removedCount = 0; allItems.forEach(item => { const button = item.querySelector('button'); if (!button) { // إذا لم يكن هناك زر، احذف العنصر item.remove(); removedCount++; return; } // فحص محتوى الزر const buttonText = button.textContent.trim(); const buttonIcons = button.querySelectorAll('i, svg, img'); const hasVisibleContent = buttonText.length > 0 || buttonIcons.length > 0; // فحص إذا كان الزر يحتوي على محتوى مرئي if (!hasVisibleContent) { console.log(`🗑️ حذف زر فارغ: ${button.outerHTML.substring(0, 100)}...`); item.remove(); removedCount++; } else { // فحص إضافي للأزرار التي تحتوي على فراغات فقط أو أرقام فقط const cleanText = buttonText.replace(/^\d+\.\s*/, '').trim(); if (cleanText.length === 0 && buttonIcons.length === 0) { console.log(`🗑️ حذف زر يحتوي على أرقام/فراغات فقط: ${buttonText}`); item.remove(); removedCount++; } } }); console.log(`✅ تم حذف ${removedCount} زر فارغ من القائمة المحمولة`); }

طرق الكشف

معايير الكشف عن الأزرار الفارغة:

  • عدم وجود زر: `!button` - عنصر li بدون button
  • نص فارغ: `buttonText.trim().length === 0` - لا يحتوي على نص
  • بدون أيقونات: `buttonIcons.length === 0` - لا يحتوي على i, svg, img
  • أرقام فقط: `cleanText.replace(/^\d+\.\s*/, '').trim()` - بعد إزالة الأرقام لا يبقى شيء
  • فراغات فقط: النص يحتوي على فراغات فقط

اختبار حذف الأزرار الفارغة

خطوات الاختبار

📋 للاختبار الفعلي:

  1. افتح الصفحة الرئيسية (index.html)
  2. افتح Console (F12) لمراقبة عملية الحذف
  3. قلل حجم الشاشة لأقل من 900px
  4. انقر على زر القائمة (☰) في الهيدر
  5. تحقق من النتائج:
    • ✅ في Console: "🗑️ حذف الأزرار الفارغة من القائمة المحمولة..."
    • ✅ في Console: "🗑️ حذف زر فارغ: ..." (لكل زر محذوف)
    • ✅ في Console: "✅ تم حذف X زر فارغ من القائمة المحمولة"
    • ❌ **لا توجد أزرار فارغة** في القائمة (مثل رقم 1 و 2)
    • ✅ **فقط الأزرار المفيدة** تظهر في القائمة
  6. تحقق من القائمة:
    • جميع الأزرار تحتوي على نص أو أيقونات
    • لا توجد مساحات فارغة في القائمة
    • القائمة مرتبة ونظيفة
