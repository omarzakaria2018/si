اختبار عناوين إحصائيات الجوال

إضافة أسماء المدن والعقارات لعناوين الإحصائيات في الجوال

التحديث المطبق

المطلوب: في إحصائيات الجوال، إظهار اسم المدينة والعقار في عناوين الأقسام، مثلما يظهر اسم العقار في الشاشات الكبيرة.

الحالة قبل التحديث بعد التحديث
بدون فلاتر إحصائيات المباني والأراضي إحصائيات المباني والأراضي
مدينة فقط إحصائيات المباني والأراضي إحصائيات الرياض
عقار فقط إحصائيات المباني والأراضي إحصائيات برج النخيل
مدينة + عقار إحصائيات المباني والأراضي إحصائيات برج النخيل - الرياض

سيناريوهات الاختبار

أمثلة على كيفية ظهور العناوين في حالات مختلفة:

بدون فلاتر

الفلاتر: الكل
إحصائيات المباني والأراضي
إحصائيات العقود
الإحصائيات المالية

مدينة محددة

الفلاتر: المدينة = الرياض
إحصائيات الرياض
عقود الرياض
الماليات - الرياض

عقار محدد

الفلاتر: العقار = برج النخيل
إحصائيات برج النخيل
عقود برج النخيل
الماليات - برج النخيل

مدينة + عقار

الفلاتر: الرياض + برج النخيل
إحصائيات برج النخيل - الرياض
عقود برج النخيل - الرياض
الماليات - برج النخيل - الرياض

التغييرات التقنية

1. إضافة منطق تحديد العنوان:

// تحديد النص المناسب للعنوان let headerText = 'إحصائيات المباني والأراضي'; if (currentCountry && currentCountry !== 'الكل') { if (currentProperty && currentProperty !== 'الكل') { headerText = `إحصائيات ${currentProperty} - ${currentCountry}`; } else { headerText = `إحصائيات ${currentCountry}`; } } else if (currentProperty && currentProperty !== 'الكل') { headerText = `إحصائيات ${currentProperty}`; }

2. تطبيق العنوان الديناميكي:

<div class="section-header"> <h3><i class="fas fa-chart-bar"></i> ${headerText}</h3> </div>

3. تطبيق نفس المنطق على جميع الأقسام:

// قسم العقود let contractsHeaderText = 'إحصائيات العقود'; if (currentCountry && currentCountry !== 'الكل') { if (currentProperty && currentProperty !== 'الكل') { contractsHeaderText = `عقود ${currentProperty} - ${currentCountry}`; } else { contractsHeaderText = `عقود ${currentCountry}`; } } // قسم الماليات let financialHeaderText = 'الإحصائيات المالية'; if (currentCountry && currentCountry !== 'الكل') { if (currentProperty && currentProperty !== 'الكل') { financialHeaderText = `الماليات - ${currentProperty} - ${currentCountry}`; } else { financialHeaderText = `الماليات - ${currentCountry}`; } }

خطوات التحقق

  1. افتح التطبيق على الجوال أو قم بتصغير النافذة
  2. اختبر بدون فلاتر - يجب أن تظهر العناوين الافتراضية
  3. اختر مدينة - يجب أن تظهر اسم المدينة في العناوين
  4. اختر عقار - يجب أن تظهر اسم العقار والمدينة
  5. جرب تركيبات مختلفة - تأكد من صحة العناوين
