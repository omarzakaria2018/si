الكود المضاف

🔧 في دالة toggleSidebar:

// إخفاء/إظهار الإحصائيات عند تبديل القائمة if (totalContainer) { if (sidebar.classList.contains('active')) { // عند فتح القائمة - إخفاء الإحصائيات totalContainer.style.display = 'none'; console.log('📊 تم إخفاء الإحصائيات عند فتح القائمة'); } else { // عند إغلاق القائمة - إظهار الإحصائيات totalContainer.style.display = ''; console.log('📊 تم إظهار الإحصائيات عند إغلاق القائمة'); } }

📱 في event listener زر القائمة:

mobileMenuBtn.addEventListener('click', function() { mobileMenu.classList.add('active'); menuOverlay.classList.add('active'); document.body.style.overflow = 'hidden'; // إخفاء الإحصائيات عند فتح القائمة const totalContainer = document.getElementById('totalContainer'); if (totalContainer) { totalContainer.style.display = 'none'; console.log('📊 تم إخفاء الإحصائيات عند فتح القائمة المحمولة'); } });

🔄 في event listeners الإغلاق:

// إظهار الإحصائيات عند إغلاق القائمة const totalContainer = document.getElementById('totalContainer'); if (totalContainer) { totalContainer.style.display = ''; console.log('📊 تم إظهار الإحصائيات عند إغلاق القائمة'); }

اختبار الميزة

خطوات الاختبار

📋 للاختبار الفعلي:

  1. افتح الصفحة الرئيسية (index.html)
  2. قلل حجم الشاشة لأقل من 900px أو استخدم الهاتف
  3. افتح Console (F12) لمراقبة الرسائل
  4. انقر على زر القائمة (☰) في الهيدر
  5. تحقق من:
    • اختفاء قسم الإحصائيات
    • ظهور رسالة في Console
    • فتح القائمة الجانبية
  6. أغلق القائمة بالنقر على (×) أو خارج القائمة
  7. تحقق من:
    • ظهور قسم الإحصائيات مرة أخرى
    • ظهور رسالة في Console

🔍 رسائل Console المتوقعة:

📊 تم إخفاء الإحصائيات عند فتح القائمة المحمولة 📊 تم إظهار الإحصائيات عند إغلاق القائمة المحمولة 📊 تم إظهار الإحصائيات عند إغلاق القائمة (خلفية) 📊 تم إخفاء الإحصائيات عند فتح القائمة 📊 تم إظهار الإحصائيات عند إغلاق القائمة
